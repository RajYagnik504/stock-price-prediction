# 📈 Stock Price Prediction System

## 🚀 Overview
This project predicts stock prices using Machine Learning (Linear Regression).

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 📊 Model Performance
- R² Score: 0.97
- MAE used for evaluation

## 📌 Features
- Data preprocessing
- Feature selection
- Model training & testing
- Visualization of actual vs predicted values

## 🎯 Conclusion
The model shows strong predictive performance and demonstrates practical application of regression in financial data analysis.
